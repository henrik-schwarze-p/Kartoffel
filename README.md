In this directory are the sources of the Sketch you downloaded.

You can compile and run the Sketch in you computer (Mac, Windows or Linux) and upload it to a Arduino Mega.

# Macintosh

You need to install brew, clang, vscode and the "C/C++" extension.

Then you need to install SDL2:

```
brew install SDL2
```

Open in vscode, and you are ready to go.

# Windows

You need to install brew, vscode and the "C/C++" extension.

Then you need to install SDL2:

```
brew install SDL2
```

Open in vscode, and you are ready to go.

# Linux

You need to install brew, vscode and the "C/C++" extension.
